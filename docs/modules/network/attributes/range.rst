network
+++++++

This represents the host range in the `CIDR <https://en.wikipedia.org/wiki/Classless_Inter-Domain_Routing>`_ notation. You can use either specify the range by it's CIDR notation or first and last IP address, but not both. If both are specified, the ``network`` attribute takes precedence.

firstIP
+++++++

This represents the first IP address in the host range.

lastIP
++++++

This represents the last IP address in the host range.
